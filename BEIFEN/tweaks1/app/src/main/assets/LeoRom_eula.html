LeoTweaks（以下简称 Leo个性化）是 Leo ROM 项目的个性化工具，可向用户提供更多的系统控制权，以不同的方式优化与个性化设置。允许用户以安全可靠的方法定制系统达到个性化的目的

• 关于捐赠
1. 个性化设置 所有功能 目前均为免费使用。
2. 捐赠完全出自你的个人意愿捐赠。

• 联网
1. Toolbar背景来源网络需要网络支持。
2. 在线更新需连接服务器,同样需要网络支持。
3. 查看开源项目等通过内置WEB浏览器联网。
4. 你可以通过禁用Leo个性化联网功能，可阻止Leo个性化访问网络。
• 免责
1. Leo个性化 所有内容资源均来自互联网，部分源代码来自 Github 的开源项目, 详细的开源项目引用列表可通过【关于】>【特别致谢】菜单查看到。
2. Leo ROM 中所涉及参考其他作者的代码，详细列表可通过【关于】>【特别致谢】菜单查看到。
2. Leo个性化 将会对用户反馈的问题和提交的建议进行修复和改善
3. 在每一个新版本中提供的功能可能会与老版本有较大出入
