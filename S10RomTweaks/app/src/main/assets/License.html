许可协议

使用LeoROM受到下列条款的制约：

一.关于捐赠

LeoROM为免费使用,但是将会受到限制.且捐赠后可以使用完整功能 捐赠仅支持一部手机
1. 个性化设置 部分功能 需要捐赠方可使用
2. 需要捐赠才可以使用的功能都有明显提示
3. 捐赠完全出自你的个人意愿捐赠

二.联网需求

1. Leo个性化部分资源以及功能需要网络支持,需联网方可使用
2. 如果你同意联网点击下一步
3. 捐赠用户在解锁后可以屏幕联网检查

三.功能差异

1. 根据所在地区以及设备版本功能上有所差异

2. 功能上的差异与捐赠金额无关

四.免责声明

1. Leo个性化 所有内容资源均来自互联网，部分源代码来自 Github 以及互联网上的开源项目, 详细的开源项目引用列表可通过【侧滑】>【开源库】查看
2. FusionLeo ROM 中所涉及参考其他作者的代码，详细列表可通过【侧滑】>【特别致谢】菜单查看到
3. 在每一个新版本中提供的功能可能会与老版本有较大出入
4. 如需参考Leo的ROM请事先通知我们
5. 禁止任何人在未征求团队同意的私自偷窃.拆解LeoROM代码
