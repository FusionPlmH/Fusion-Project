FusionLeo（以下简称 Leo个性化）是Leo ROM 项目的个性化工具，可向用户提供更多的系统控制权，以不同的方式优化与个性化设置。允许用户以安全可靠的方法定制系统达到个性化的目的

• 关于捐赠
1. 个性化设置 部分功能 需要捐赠方可使用。
2. 捐赠完全出自你的个人意愿捐赠。

• 联网
1. Toolbar背景来源网络,需要网络支持。
2. 在线更新需连接服务器,同样需要网络支持。
3. 查看开源项目等通过内置WEB浏览器联网。
4. 你可以通过禁用Leo个性化联网功能，可阻止Leo个性化访问网络。
• 免责
1. Leo个性化 所有内容资源均来自互联网，部分源代码来自 Github 的开源项目, 详细的开源项目引用列表可通过【侧滑】>【特别致谢】菜单查看到。
2. FusionLeo ROM 中所涉及参考其他作者的代码，详细列表可通过【关于】>【特别致谢】菜单查看到。
2. Leo个性化 将会对用户反馈的问题和提交的建议进行修复和改善
3. 在每一个新版本中提供的功能可能会与老版本有较大出入

• 注意事项
1. 在刷机完成之后，测试满意之后在进行数据恢复及软件安装。
2. 在使用过程中如果因个人修改导致的问题我们不予回应。
个性化设置 通过开发者个人以及测试团队的测试，对于产品本身存在的问题或者技术原因无法改善的问题，将在使用中或使用前主动向用户进行明确的告知
小更新我们会OTA推送更新,大更新不推送OTA需要自行下载卡刷包

2. QQ群仅提供机油之间的讨论资源共享等，不涉及最新官方信息发布
