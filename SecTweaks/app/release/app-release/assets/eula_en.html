FusionLeo (hereafter referred to as Leo Personalization) is a personalized tool for the FusionLeo ROM project that provides users with more control over the system and optimizes and personalizes them in different ways. Allow users to customize the system in a safe and reliable way to achieve personalization

• About donations
1. Personalization Some features need to be used by donors.
2. Donations are entirely due to your personal willingness to donate.

• Networking
1. Toolbar Background Source network requires network support.
2. Online updates require a connection to the server and network support is also required.
3. View open source projects and other networking through the built-in WEB browser.
4. You can prevent Leo from personalizing access to the network by disabling Leo's personalized networking features.
